# Portfolio CV - Yacouba Ouedraogo

Portfolio minimaliste en français, prêt pour GitHub Pages.

La version actuelle inclut la photo de graduation et un résumé du mémoire de fin d’études.

## Contenu
- Profil
- Coordonnées
- Compétences
- Parcours académique
- Mémoire de fin d’études
- Expériences / stages
- Connaissances techniques
- Outils
- Centres d’intérêt

## Lancer en local
Ouvrir simplement `index.html` dans le navigateur, ou servir le dossier avec un petit serveur statique.

Exemple:

```bash
cd /Users/albarka/Desktop/Projects/yacou
python3 -m http.server 8000
```

Puis ouvrir `http://localhost:8000`.

## Publier sur GitHub Pages
1. Créer un dépôt GitHub.
2. Envoyer ces fichiers à la racine du dépôt.
3. Dans les paramètres du dépôt, activer GitHub Pages sur la branche principale et la racine `/`.
4. Attendre la génération du site.

## Modifier le contenu
Le contenu principal est dans `script.js`. Il suffit de remplacer le texte par les informations finales du CV si besoin.

## Note confidentialité
Le numéro de carte de séjour n’a pas été repris dans le site pour éviter d’exposer un identifiant sensible publiquement.
